##  Features
- **Modern UI/UX:** Stunning dark-mode design with Glassmorphism effects and smooth transitions.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop screens.
- **Dynamic Projects Section:** Interactive project cards with custom images and technology badges.
- **Optimized Performance:** Built with Next.js 15 and Turbopack for ultra-fast loading and smooth navigation.
- **Interactive Sections:**
  - **About Me:** Professional background and career highlights.
  - **Experience:** Detailed timeline of professional work.
  - **Projects:** Showcasing live applications with GitHub integration.
  - **Skills:** Categorized technical expertise with proficiency indicators.
  - **Contact:** Integrated communication channels for collaboration.

## 🚀 Getting Started

To run this project locally, follow these steps:

### 1. Clone the repository
```bash
git clone https://github.com/zupeirr/FUTURE_FS_01.git
cd FUTURE_FS_01
```

### 2. Install dependencies
```bash
pnpm install
```

### 3. Run the development server
```bash
pnpm dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).
